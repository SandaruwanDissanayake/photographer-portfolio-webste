# 📸 Photographer Portfolio Website

An elegant, modern, and fully responsive portfolio website designed for professional photographers to showcase their work, services, and expertise. This project combines a clean frontend design with a PHP backend for dynamic functionality.

---

## ✨ Features

- **📱 Fully Responsive Design:** Clean layout optimized for all screen sizes including desktops, tablets, and mobile devices.
- **🖼️ Interactive Photo Gallery:** Beautifully organized sections to showcase photography categories (e.g., Wedding, Portrait, Landscape).
- **💻 Smooth User Experience:** Built with clean HTML5, CSS3, and modern JavaScript for smooth scrolling and animations.
- **🐘 Dynamic Backend:** Powered by PHP to handle background operations, page routing, or form actions efficiently.
- **📩 Contact Form:** A functional contact section allowing potential clients to reach out and book sessions easily.

---

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript (Vanilla JS)
- **Backend:** PHP
- **Styling & Icons:** FontAwesome / Custom CSS
- **Environment:** Compatible with Apache server (XAMPP / WAMP / Local)

---

## 🚀 Getting Started

Follow these steps to set up and run the project locally on your machine.

### Prerequisites

To run this project, you need a local web server that supports PHP:
- [XAMPP](https://www.apachefriends.org/) (Recommended)
- WAMP
- MAMP

### Installation & Setup (Using XAMPP)

1. **Clone the Repository:**
   Clone this project directly into your XAMPP `htdocs` directory:
